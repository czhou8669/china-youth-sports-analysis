# 全国青少年体育赛事数据分析

基于 2023-2024 年全国青少年体育赛事数据（共 5325 条），结合青少年体育消费行为研究视角，进行数据清洗、探索性分析与可视化，并引入人口、经济维度进行多变量关联分析，专项深入长三角区域特征研究。

## 项目背景

本项目数据来源于**全国群众体育赛事数据库**。分析工作结合上海体育大学经济管理学院吴际教授团队课题《上海市青少年体育投资消费行为研究》的研究积累，从赛事供给侧数据中理解青少年体育参与和消费行为的规律。

## 数据说明

| 字段 | 说明 |
|------|------|
| 数据来源 | 全国群众体育赛事数据库 |
| 数据量 | 5325 条（2023 年 2742 条 + 2024 年 2583 条） |
| 覆盖范围 | 全国 31 省/市/自治区，39 个运动项目 |
| 核心字段 | 省市区、赛事级别、运动项目、赛事规模、开始/结束时间 |

> ⚠️ **数据质量问题**：经交叉核验，该数据库存在省级覆盖严重不足的问题（广东、江苏、四川三省偏差尤为突出），部分字段（直辖市赛事级别分类、运动项目名称噪音等）需人工清洗。本分析中的省际排名和密度比较结论需谨慎对待。

## 核心发现

| # | 结论 |
|---|------|
| 一 | **地区分布**：东部高度集中，上海、山东、浙江合计占全国约 40% |
| 二 | **直辖市分化**：上海走"广度覆盖"路线（767 场，中位数 50 人），北京走"高度精英"路线（491 场，中位数 300 人，含高尔夫、马术等高消费项目） |
| 三 | **消费结构**：球拍类（网羽乒）规模偏小（均值 296 人）但客单价高；团队球类（足篮排）双重领先（均值 713 人，足球 537 场全国第一） |
| 四 | **驱动力分层**：个人偏好/学校要求（内生）、家长经济（决策）、空间状况（制约）三层递进 |
| 五 | **新兴运动**：攀岩/街舞已入奥运，场馆壁垒差异（冰球极高 vs 街舞极低）决定复制速度 |
| 六 | **赛事节奏**：高峰 7-8 月（暑假），低谷 1-2 月（寒假），与学校日历高度绑定 |
| 七 | **人口大省**：赛事数量与人口规模不成正比，广东（0.2 场/百万人）等多为数据盲区 |
| 八 | **经济关联**：人均 GDP 与赛事密度相关系数 r=0.73，呈较强正相关 |
| 九 | **边疆特征**：西藏、新疆等地赛事密度虚高，主要为政策驱动而非市场自发 |
| 十 | **长三角**：以 16.9% 人口贡献 31.2% 赛事，密度为全国 1.9 倍，但平均规模更低（小规模精品化） |
| 十一 | **长三角内部**：上海/浙江以市级社区赛事为主（市场化），江苏省级占比 60%（行政驱动），结构分化明显 |

## 使用方式

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

> **中文字体**：代码中 `config.py` 的 `setup_chinese_font()` 函数会自动适配 macOS（PingFang/STHeiti）、Windows（微软雅黑）、Linux（文泉驿），无需手动配置。

### 2. 放入数据文件

将原始 Excel 文件（名为 `全国群众体育赛事、青少年体育赛事、马拉松赛事信息.xlsx`）放入 `data/` 目录：

```
data/全国群众体育赛事、青少年体育赛事、马拉松赛事信息.xlsx
```

### 3. 一键运行全流程

```bash
python main.py
```

运行步骤：
1. 数据清洗 → 输出 `data/青少年体育赛事_清洗后.csv`
2. 核心分析 → 终端打印分析结果
3. 人口经济分析 → 终端打印 correlation 和四象限分析
4. 长三角专题分析 → 终端打印
5. 生成 11 张可视化图表 → 输出到 `output/charts/`

### 4. 单独运行某一步骤

```bash
python main.py --clean-only    # 仅执行数据清洗
python main.py --viz-only      # 仅生成图表（需先完成清洗）
python main.py --help          # 查看帮助
```

## 初期建议（精简版）

1. **项目类型与营销匹配**：球拍类走高客单价精品路线（装备展销+培训引流），团队球类走青训体系入口+品牌曝光
2. **上海双层结构**：在社区赛事基础上叠加标杆性精品赛事，打造"大众铺底+精品拔高"
3. **新兴项目分赛道**：街舞低门槛快速复制，击剑走精品化私校渗透，冰球需场馆资源
4. **地区差异化**：广东/江苏区分"真实空白"和"统计盲区"；内陆大省先做基础设施再引赛事
5. **长三角协同**：统一赛事日历、联合申办、积分互通；江苏需先推赛事市场化改革
6. **季节性运营**：寒假布局室内赛事填补空白，暑期需差异化定位
7. **体教融合**：以校园赛事为入口，构建"校→区→市→省→国家"金字塔

## 数据清洗过程

| 步骤 | 内容 |
|------|------|
| 1 | 删除无效列：官网、微博、微信、赞助级别等 16 列全部为空，直接删除 |
| 2 | 统一地区字段：删除省/市编码列，保留中文名称列 |
| 3 | 修复赛事级别异常值："C（属地办赛）"→市级，"A(A1)"→国家级 |
| 4 | 时间字段处理：转 datetime 格式，计算持续天数、提取月份 |
| 5 | 空值处理：87 条运动项目空值填为"未知"，赛事规模转数值型 |
| 6 | 噪音过滤：运动项目分析时过滤含引号的活动全称字段 |
